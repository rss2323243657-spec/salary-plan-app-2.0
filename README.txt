📱 薪酬计划表 - 手机App使用指南
================================

🌟 功能特点：
- 跨年份查看多个月份数据
- 自定义收入/支出/负债明细
- 选择性批量填充固定项
- 编辑实际支出并对比规划
- 数据自动保存到手机

📲 如何在手机上使用：

方法1：直接访问（推荐）
------------------------
1. 用手机浏览器（Chrome/Safari）打开这个链接
2. 浏览器会提示"添加到主屏幕"
3. 点击"添加"即可安装为App

方法2：手动添加
----------------
【iPhone (Safari)】
1. 打开 Safari 浏览器
2. 访问本页面
3. 点击底部分享按钮
4. 选择"添加到主屏幕"
5. 点击"添加"

【Android (Chrome)】
1. 打开 Chrome 浏览器
2. 访问本页面
3. 点击右上角菜单(三个点)
4. 选择"添加到主屏幕"
5. 点击"添加"

🎯 使用技巧：
1. 支持离线使用 - 安装后可无网络使用
2. 数据自动保存 - 关闭App后数据不丢失
3. 全屏体验 - 像原生App一样使用
4. 可分享 - 发送链接给家人共同查看

⚠️ 注意事项：
- iOS设备需要 Safari 浏览器
- Android设备需要 Chrome 浏览器
- 首次访问需要网络，之后可离线使用
- 定期点击"保存当前数据"按钮备份

📧 支持：
如有问题，请检查：
1. 浏览器是否最新版本
2. 是否使用 HTTPS 访问（本地开发可用HTTP）
3. 是否允许浏览器安装应用
